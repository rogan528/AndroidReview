# Java基础
## 1、 JDK和JRE的区别
 1、JDK(Java Development Kit)是Java开发工具包,JRE(Java Runtime Environment)Java运行环境
 
 2、JDK包含JRE,JRE包含JVM
## 2、 标识符定义
 由26个英文字母大小写、数字、0-9、符号_  $组成，并且只能以字母、下划线、美元符号开头。
